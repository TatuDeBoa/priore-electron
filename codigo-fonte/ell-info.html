1. DEFINIÇÃO

Um assunto importante da termodinâmica é o equilíbrio de fases, com aplicações tecnológicas de grande interesse industrial. Por exemplo, a partir do estudo de sistemas de equilíbrio líquido-vapor foi possível o desenvolvimento de uma das mais importantes operações unitárias, a destilação. Outro caso é o Equilíbrio Líquido-Líquido (ELL), particularmente interessante para operações de extração, relacionadas muitas vezes com produtos de alto valor agregado.

A definição termodinâmica para equilíbrio de fases é a isofugacidade, em que ambas as fases possuem a mesma fugacidade (SMITH, 2009). Porém, desenvolver equacionamentos a partir dessa definição é complicada e trabalhosa, pois necessita de modelos para Energia de Gibbs em Excesso, que podem ou não serem dependentes de dados experimentais de interação binários. Assim, foi desenvolvido um novo método para avaliar e predizer um sistema ELL.

Esse método é o ajuste de dados experimentais de equilíbrio para um determinado sistema em uma equação característica, que representa o comportamento característico sistema ELL (HAN et al., 2014). Este é esquematizado na Figura 1, em que alfa (α) e beta (β) são os componentes do sistema e os valores de concentração estão em porcentagem massa/massa.

2. EXTRATÉGIA DE RESOLUÇÃO UTILIZADA

Zero possui duas áreas na seção ELL, uma destinada para ajuste dos dados experimentais em um dos modelos cadastrados e outra para predição das concentrações das fases em equilíbrio a partir de uma mistura, com a escolha de uma equação característica. Essa divisão foi realizada pelo fato de serem processos separados e independentes. Primeiro é feito a análise do ajuste dos dados experimentais, depois, com uma expressão definida, é possível executar a predição.

Para ajuste, é feita a mudança de variáveis e então regressão linear múltipla. Este processo é realizado por uma rotina interna (classe Regressor), que recebe apenas os valores numéricos de yi e xn,i e calcula os parâmetros, o coeficiente de determinação, erro padrão dos estimadores e outras definições estatísticas, que não são exibidas na seção ELL para evitar poluição visual.

A predição envolve uma rotina numérica iterativa, pois ao acoplar a equação do modelo escolhido com o balanço de massa é gerado um sistema de equações não lineares. Para resolução deste, Zero utiliza um algoritmo específico para aplicar o método de Newton (REF) e resolver o sistema. Tal método necessita de um estimador inicial, do cálculo da matriz Jacobiana e da resolução de um sistema linear simples para encontrar o incremento do estimador de iteração. O procedimento possui rápida convergência, mas é fortemente influenciado pela qualidade do estimador inicial. Esta aplicação utiliza como padrão estimadores iniciais que geralmente resultam na convergência do sistema, mas permite também que o usuário altere os estimadores caso deseje ou ocorra algum erro.

3. PROCEDIMENTO EXPERIMENTAL POSSÍVEL

A seção ELL se assemelha com a DTP no caso de procedimento experimental, pois a própria concepção do processo está relacionada com dados laboratoriais. Com este norte, Zero apresenta uma interface intuitiva que resolve todas as necessidades no mesmo ambiente. Realiza o ajuste de uma equação, calcula alguns parâmetros estatísticos, gera gráficos e permite a predição de concentração no equilíbrio de fases de uma mistura.

Uma forma de se obter dados de equilíbrio é a partir do teste turbidimétrico, utilizados em sistemas aquasos bifásicos (SAB), de grande interesse por serem não tóxicos e com capacidade de extração de susbstâncias com alto valor agregado (HAN et al., 2014). O procedimento é simples e necessita dos componentes do sistema, um recipiente para alocação do sistema (por exemplo, tudo de ensaio) e balança analítica para medição das massas adicionadas.

4. MODO DE UTILIZAÇÃO DOS PROCEDIMENTOS DISPONÍVEIS

A metodologia de ajuste de dados é simples, necessita que o usuário informe os dados experimentais obtidos. É de responsabilidade do mesmo que identifique para o seu caso qual é o componente alfa (forma a fase pesada) e qual o componente beta (forma a fase leve), ou seja, em função de qual componente irá descrever o outro. Para consultar as equações dos modelos, existe um link "Exibir Equações dos Modelos", localizado na parte superior direita, logo abaixo do nome da seção, cujo clique exibe ou oculta as expressões dos modelos. A entrada de dados é semelhante aos espaços das planilhas eletrônicas, o que torna mais intuitivo e fácil a inserção deles, pois podem ser copiados diretamente de uma planilha e colados nos espaços do programa. Após informar os valores disponíveis, a resolução é informada depois após clicar no botão "Executar Cálculos", que exibirá na tela as respostas e os gráficos com os dados experimentais e as funções ajustadas.

Para predição, o procedimento de inserção de dados é diferente. Primeiro é preciso escolher um modelo entre os três disponíveis e depois alocar os parâmetros dele nos espaços abaixo da lista de seleção. O próximo passo é informar a Razão de Fases, que é a razão entre a massa de fase de topo e a massa total da mistura (soma da massa das duas fases). Por fim, é preciso inserir a composição da mistura que formará duas fases (Ponto de Mistura). O usuário pode informar quantos pontos de mistura desejar, através do formulário que contêm lacunas para concentrações de alfa e beta e um botão de adição, que grava as informações digitadas e as exibe ao lado.

Há também a possibilidade da alteração dos estimadores iniciais padrões do programa. Para isso é preciso clicar no link "Configurações Avançadas", que abrirá um novo espaço para que se informe os estimadores iniciais da concentração de alfa na fase superior (rica em beta) e da concentração de alfa na fase inferior (rica em alfa). O usuário pode alterar esses estimadores por vontade própria ou para consertar um erro que convergência. Quando um erro deste tipo ocorre, no lugar dos valores numéricos é exibido a mensagem "NaN".

Finalizadas as inserções de dados à aplicação, clicar no botão "Executar Cálculos" dispara o processo de resolução e assim que terminado, serão exibidas mensagens no espaço abaixo do botão com as respostas da predição, juntamente com um gráfico com a linha de amarração dos pontos de mistura e a curva gerada pela equação escolhida.
